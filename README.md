## Project Overview

This project performs end-to-end analysis of customer shopping behavior
to generate actionable business insights. The workflow includes data
cleaning, SQL analysis, exploratory data analysis, and interactive
dashboard creation.

The goal is to help businesses understand purchasing patterns, customer
segments, and product performance to improve decision-making.
